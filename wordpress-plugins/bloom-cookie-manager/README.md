# 🍪 bloom-cookie-manager

**Gestión avanzada de cookies, consentimiento y sesiones de usuario para WordPress.**

## 📋 Descripción
Plugin de gestión de cookies y consentimiento que controla de forma granular el almacenamiento y ciclo de vida de cookies del sitio, cumpliendo con regulaciones de privacidad. Desarrollado a lo largo de 10+ releases para adaptarse a las necesidades cambiantes de una plataforma de alto tráfico.

## 🚀 Características
- Banner de consentimiento personalizable con categorías de cookies (esenciales, analíticas, publicitarias)
- Panel administrativo para configurar políticas de retención y caducidad
- Bloqueo selectivo de scripts de terceros según el consentimiento del usuario
- Integración con el sistema de caché del sitio

## 💻 Tecnologías
- PHP 7.4+, WordPress Hooks API
- JavaScript, CSS3
