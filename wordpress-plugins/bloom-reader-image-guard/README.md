# 🖼️ bloom-reader-image-guard

**Protección anti-descarga de imágenes del lector de manga para WordPress.**

## 📋 Descripción
Plugin que implementa múltiples capas de protección para prevenir la descarga directa y el scraping de las imágenes del reader de manga. Utiliza técnicas de ofuscación del DOM, bloqueo de eventos de contexto y protección a nivel de servidor.

## 🚀 Características
- Bloqueo de clic derecho y menú contextual sobre las imágenes del reader
- Prevención de arrastrar y soltar imágenes
- Ofuscación de URLs directas de las imágenes en el DOM
- Protección contra scraping automatizado

## 💻 Tecnologías
- PHP 7.4+, WordPress Hooks API
- JavaScript (ES6+)
