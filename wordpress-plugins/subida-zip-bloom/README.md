# 📦 subida-zip-bloom

**Módulo de carga masiva de capítulos manga vía archivos ZIP para WordPress.**

## 📋 Descripción
Plugin que automatiza la extracción, procesamiento y publicación de capítulos de manga directamente desde archivos ZIP. Pensado para equipos de scan que necesitan subir grandes volúmenes de imágenes de forma rápida y eficiente. Desarrollado a lo largo de 9 releases (v3.3 → v3.5.4).

## 🚀 Características
- Upload de archivos ZIP con extracción automática de imágenes
- Procesamiento y ordenamiento secuencial de páginas
- Creación automática del post de capítulo en WordPress con metadatos asociados
- Validación de formatos de imagen y límites de tamaño
- Interfaz administrativa para gestionar las cargas en cola

## 💻 Tecnologías
- PHP 7.4+, WordPress Media API
- ZipArchive, File System API
