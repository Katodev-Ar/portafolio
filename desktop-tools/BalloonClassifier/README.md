# 🎈 BalloonClassifier

**Clasificador inteligente de globos de texto en páginas de manga.**

## 📋 Descripción
Herramienta de visión artificial que detecta y clasifica automáticamente los globos de texto (speech balloons) en páginas de manga, facilitando los flujos de traducción y limpieza de imágenes.

## 🚀 Características
- Detección automática de globos de texto en imágenes de manga
- Clasificación por tipo (diálogo, pensamiento, narración, efectos de sonido)
- Exportación de coordenadas y regiones detectadas
- Pipeline de 4 versiones iterativas con mejoras de precisión

## 💻 Tecnologías
- Python 3, OpenCV / IA
