# 🖌️ WatermarkRemove

**Suite de eliminación y recoloración automática de marcas de agua en imágenes manga.**

## 📋 Descripción
Colección de herramientas (5 versiones de producción + 1 autodetector) para la eliminación y recoloración automatizada de marcas de agua en páginas de manga. Cada versión incorpora mejoras progresivas en la precisión de detección y el rendimiento del procesamiento por lotes.

## 🚀 Características
- **Procesamiento por lotes** de carpetas completas de imágenes
- **Múltiples estrategias de detección:** por coordenadas fijas, por patrones de color y autodetección inteligente
- **Motor de recoloración** que muestrea el color de fondo circundante y lo aplica sobre la marca de agua
- **Pipeline de 5 versiones** con mejoras iterativas de precisión

## 💻 Tecnologías
- Python 3, Pillow/PIL
- Interfaz gráfica básica (UI)
