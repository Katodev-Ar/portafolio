❓ Guía de uso
1. Cargar imágenes
Staff: usa Selección de Proyecto → elige serie y capítulo → Nuevo Proyecto (Drive).
Usuario: usa Cargar imágenes desde tu PC.
2. Seleccionar globos
Arrastra para crear recuadros azules sobre cada globo.
Click derecho para borrar un recuadro. Ctrl+rueda para zoom.
3. OCR → Generar OCR
Selecciona el idioma del manhwa y pulsa Generar. El texto aparece numerado en Transcripción.
Pulsa un número para resaltar el globo en la imagen, y viceversa.
4. Glosario (opcional)
Agrega nombres y contexto en la pestaña Glosario. Gemini lo usará al traducir.
5. Traducir
Elige idioma destino y pulsa Traducir. Costo: 1 crédito cada 600 caracteres.
6. Etiquetas (D:, P:, N:...)
Clic en cualquier etiqueta de la barra para insertarla al inicio de la frase activa.
Si hay otra etiqueta, se reemplaza automáticamente. También puedes usar F1–F12.
7. Buscar / Reemplazar
Ctrl+F abre el buscador en la pestaña activa (Transcripción o Traducción).
8. Exportar
JSON de coordenadas compatible con el script TypeSetter Pro de Photoshop (1 crédito).
TXT: transcripción + traducción. Clean: imágenes limpias (2 créditos/imagen).
